# SenaAvicola_SENA

Sistema Avícola para el SENA - Gestión de granja avícola.

## Descripción
Aplicación web para la gestión de una granja avícola, con roles de administrador, aprendiz y visitante. Incluye funcionalidades como registro de lotes, producción, stock, ventas, muertes, vacunas, y estadísticas con gráficas.

## Tecnologías
- HTML
- CSS
- JavaScript
- Chart.js

## Cómo ejecutar
1. Clona el repositorio.
2. Abre index.html en un navegador o usa un servidor local como `python -m http.server 8000`.
